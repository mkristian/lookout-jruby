# Lookout::Jruby

JRuby hacks/compatibility code for YAML/Configatron

## Installation

Add this line to your application's Gemfile:

    gem 'lookout-jruby'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lookout-jruby

## Usage

require 'lookout/jruby' in your project
